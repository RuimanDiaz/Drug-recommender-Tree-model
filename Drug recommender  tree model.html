Machine learning for Drug Patient Recommender¶

the aim for this report is to create a model that can predict the type of drug that each patient should use according to this data, in this case we will be using Decision trees a non-parametric supervised learning method used for classification and regression. The goal is to create a model that predicts the value of a target variable by learning simple decision rules inferred from the data features. A tree can be seen as a piecewise constant approximation.
